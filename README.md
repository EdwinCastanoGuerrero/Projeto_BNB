# Clone BRB

Um projeto de clone da interface do BRB feito com React e Vite. Serve como estudo de componentes React, Bootstrap e animações CSS.

## Tecnologias

- React 19
- Vite
- Bootstrap 5
- MDI Icons

## Como usar

`bash
npm install
npm run dev
`

`

## Componentes

**Navbar** - Menu superior com logo, dropdown e ícones. Tem efeitos de hover e é responsivo.
